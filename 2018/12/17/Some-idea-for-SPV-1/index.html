Some idea for SPV(1)

十二月 17 2018

What is SPV ?

SPV, also known as simple payment verification, which is created by bitcoin. It’s used for verification of transation in light client which don’t have enough space for full chain.

How does it works ?

In Bitcoin, every block has a structure called header. You can find the it in my period blog. You can download the whole chain by only downloading the headers.

When you wants to a transation, you can just need to know the height of the block and the index of the transation. With the merkle tree in the headers, you can verify any transation you need.

The key inside the SPV

The key of SPV is simple, use the least information to rebuild the whole chain. So Bitcoin use merkle to store the informations. To build the whole chain, we need to include previous block. And to verify the block, we have nonce to verify the POW.

So the essence of the SPV is using a little information to build the chain, store the messages and do the verification.
