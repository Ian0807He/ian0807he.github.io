Coinbase

作为一个Bitcoin从开发的最初版本就存在的东西，coinbase是一个交易信息载体的绝佳选项，你可以有限的空间里记录下交易的一些细节信息，比如支持Unlimited，Segwit，8MB中的那一个

查找方法也很简单，只需要查找一个 tx 中 pos等于 0 的 scriptSig 就可以了，然后将16进制转化为ascii就可以看到里面的内容了
