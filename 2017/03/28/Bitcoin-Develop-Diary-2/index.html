Bitcoin Trascation (1)

Input Addres And Output Address

在比特币的思想中，我们知道，所有的交易都是可查的，那么Bitcoin是通过什么方式来是过去的交易变的可查询的呢？答案很简单，用一个类似双向链表的方式来存储交易信息。

首先要明确一个概念，就是这一笔交易的输入是上一笔交易的输出（在非coinbase交易中）。第一笔交易，也就是coinbase交易是不存在输入的，因为是挖矿，那么挖出来的奖励就存在了矿工的钱包里，那么想花钱，那么这些钱就来自上一笔交易的输入（挖矿奖励），那么下一个交易的输入就来自这笔交易的输出也就不难理解了。

那么继续，如果说，txout中的txout_id是用来标示这一笔交易的输出，txout_scriptPubKey表示的是该输出的输出地址，那么txin中的txout_id就是用来标示上一笔交易的输出。那么如果想查找上一笔交易的信息，就可以通过这个txout_id来查找了，该输入的输入地址就是上一次输出的地址txout_scriptPubKey了。
